Simulación de fluidos inmiscibles usando el método de Lattice-Boltzmann

Se implementa el método de Lattice-Boltzmann para modelar el comportamiento
de fluidos inmiscibles con diferentes densidades. Conjuntamente, se explica la
implementación y desarrollo del método de Lattice-Boltzmann para simular fluidos.
